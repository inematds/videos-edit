# Forja Reel — a skill

Meta-skill para Claude Code: pega um vídeo bruto (com cortes, silêncios e erros) e devolve um reel profissional — corte limpo, motion graphics, B-roll real, legendas e SFX — sem você abrir um editor de vídeo. Em vez de ser um editor pronto, ela te entrevista e **gera o seu próprio editor**, com a sua identidade.

## Baixar
O arquivo pronto para o Claude Code está em [`download/forja-reel.skill`](download/forja-reel.skill).
